# 单元测试成果物说明

本目录对应课程设计中的单元测试成果物。测试代码集中放置在：

`backend/src/test/java/com/shop/unit/`

## 设计方法

| 设计方法 | 覆盖内容 |
| --- | --- |
| 等价类划分 | 合法/非法用户名、密码、手机号、商品字段、收货信息 |
| 边界值分析 | 用户名长度、密码长度、搜索关键字长度、购物车数量、商品价格、库存、地址长度 |
| 判定表法 | 注册重名、登录失败次数、商品上下架、购物车库存与累加数量 |
| 状态迁移法 | 登录失败到账号锁定、锁定期内禁止登录 |
| 错误推测法 | 空字符串、全空格、空对象、越界数据 |

## 测试类说明

| 测试类 | 主要验证点 |
| --- | --- |
| `UserValidationUnitTest` | 用户注册/登录字段规格 |
| `UserServiceUnitTest` | 注册重名、密码一致性、密码加密、登录失败计数、账号锁定 |
| `ProductServiceUnitTest` | 商品搜索关键字、分页兜底、商品详情上下架判断 |
| `CartServiceUnitTest` | 加购数量、库存、累加、修改数量、删除权限 |
| `EntityValidationUnitTest` | 商品管理字段、订单收货信息、购物车数量实体约束 |

## 运行方式

在 `backend` 目录执行：

```bash
mvn test
```

只运行本成果物中的单元测试：

```bash
mvn -Dtest='com.shop.unit.*Test' test
```
